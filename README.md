# empathy-onboarding

Скилл для Claude Code / Cursor. Онбордит сотрудника компании двумя фазами:

1. **Контекст роли** — короткое интервью (роль, метрика, задачи, AI Mandate компании) → `COMPANY.md`
2. **Talent Q → архитектура ассистента** — анализ психометрического отчёта Talent Q, выявление сильных сторон и слепых зон, генерация компенсирующих поведений агента → `AGENT-PERSONALITY.md`

## Установка

В Claude Code / Cursor в рабочей папке:

```
install skill from https://github.com/<org-or-account>/empathy-onboarding
```

*(ссылка станет рабочей после публикации репозитория — на 07.2026 файлы существуют только локально)*

## Требования

- Отчёт Talent Q в любом формате (PDF/текст/скриншот) — опционально для Фазы A, обязательно для Фазы B
- Список правил AI Mandate компании (если есть) — передать фасилитатору сессии для вставки в Фазу A

## Приватность

Результаты Talent Q и файлы `COMPANY.md` / `AGENT-PERSONALITY.md` остаются локально у пользователя. Скилл (эта логика/промпты) — публичный, персональные данные сотрудников в репозиторий никогда не попадают.
